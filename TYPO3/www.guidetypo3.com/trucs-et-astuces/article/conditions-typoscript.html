Conditions typoscript

Publié le 15/11/2010 par haythoum

Il est possible d'ajouter des conditions au niveau du code typoscript dans TYPO3, ces conditions sont à mettre dans la partie Constants ou Setup.

Voici la liste non exhaustive des conditions typoscript :

globalVars

La comparaison s'effectue toujours sur les variables "var1", "var2", ... Dans l'exemple ci-dessous, la comparaison est effectuée sur l'id de la page en cours.

Syntaxe

[globalVar= var1=value, var2<value2, var3>value3, ...]

Exemple

[globalVar = TSFE:id = 10]
variable = valeur
[end]

PIDinRootline

Condition sur l'emplacement de la page dans l'arborescence, ceci vérifie si l'une des valeurs corresponds à des PID (pages-uid) de l'arborescence

Syntaxe

[PIDinRootline = pages-uid, pages-uid, ...]

Exemple

[PIDinRootline = 10, 12]
variable = valeur
[end]

treeLevel

Condition sur le niveau de la page dans l'arborescence, ceci vérifie si le dernier élément de l'arborescence correspondant à un niveau entre les valeurs de "treeLevel".
Niveau = 0 est la "racine" d'un site web.
Niveau = 1 est le premier menu

Syntaxe

[treeLevel = levelnumber, levelnumber, ...]

Exemple

[treeLevel = 3, 4]
variable = valeur
[end]

hostname

Les valeurs sont comparées avec le nom de la hôte renvoyé par la fonction prédéfinie getenv("REMOTE_ADDR") de PHP.

Syntaxe

[hostname = hostname1, hostname2, ...]

Exemple

[hostname = www.guidetypo3.com]
variable = valeur
[end]

browser

Ceci vérifie le navigateur web utilisé pour l'affichage de la page et spécifié des conditions sur quelques navigateurs comme montre l'exemple ci-dessous.

Syntaxe

[browser = browser1, browser2, ...]

Exemple

[browser = netscape, opera]
variable = valeur
[end]

system

Ceci vérifie le système d'exploitation utilisé, cela permet de spécifié des conditions sur quelques systèmes d'exploitations comme le Mac OS par exemple.

Syntaxe

[system= system1, system2]

Exemple

[system= mac]
variable = valeur
[end]


Pour trouver d'autres exemples d'utilisation des conditions typoscript dans TYPO3, veuillez visiter le TSref (Conditions).

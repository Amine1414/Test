Check a website with google

Publié le 04/02/2012 by  haythoum

La vérification d’un site TYPO3 sur Google se fait à l’aide de la balise meta google-site-verification, cette balise se met sur la page de plus haut niveau de votre site afin de confirmer que vous en êtes propriétaire dans le cadre des Outils pour les webmasters. Voici un code typoscript a mettre dans la partie SETUP de votre template typoscript qui permet d'insérer cette balise dans la partie HEAD : 


  1. page.headerData.100 = TEXT
  2. page.headerData.100.insertData=1
  3. page.headerData.100.wrap = <meta name="google-site-verification" content="|" />
  4. page.headerData.100.value = votre-code-de-verification-google
